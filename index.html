📖 Firebase 설정 방법 (처음 사용 시 펼쳐서 확인)

1단계: Firebase 프로젝트 생성

  1. Firebase 콘솔 접속 (구글 계정 필요)
  2. 프로젝트 추가 클릭 → 프로젝트 이름 입력 (예: task-tracker)
  3. Google Analytics는 사용 안 함 선택 (필수 아님)

2단계: Realtime Database 활성화

  1. 좌측 메뉴 빌드 → Realtime Database
  2. 데이터베이스 만들기 클릭
  3. 위치: asia-southeast1 또는 가까운 곳 선택
  4. 보안 규칙: 테스트 모드로 시작 선택 (나중에 보안 강화 권장)
  5. 생성 후 화면 상단의 URL을 복사 (https://...firebaseio.com 형태)

3단계: 이 화면에 입력

  1. 위에서 복사한 URL을 Firebase Project URL에 붙여넣기
  2. 팀 ID는 팀원들끼리 합의한 동일한 문자열 (예: service-ops-2026)
  3. 연결 / 동기화 시작 클릭

4단계: 팀원에게 공유

  • 같은 GitHub Pages URL을 모두 사용
  • 이 설정 화면에서 같은 Firebase URL + 같은 팀 ID 입력하면 즉시 같은 데이터를 봄
  • 한 명이 추가/수정하면 다른 사람 화면에 실시간 반영

⚠️ 보안 주의

  • 테스트 모드는 30일 후 자동 차단됩니다. Firebase 보안 규칙을 적절히 설정하세요.
  • 이 URL/팀 ID를 아는 사람은 누구나 데이터를 보고 수정할 수 있으니 공개하지 마세요.
  • 민감한 정보(개인정보, 비밀번호 등)는 저장하지 마세요.
